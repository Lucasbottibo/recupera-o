Principais Comandos HTML e CSS

Comandos HTML

<!DOCTYPE html>

Define o tipo de documento e a versão do HTML. Essencial para que o navegador saiba como renderizar a página.

<html>

O elemento raiz que envolve todo o conteúdo da página HTML.

<head>

Contém metadados, como o título da página, links para CSS e scripts.

<body>

Inclui o conteúdo visível da página, como texto, imagens e outros elementos.

<a>

Cria links para outras páginas ou recursos. Exemplo: <a href="https://example.com">Link</a>.

<img>

Insere imagens na página. Exemplo: <img src="imagem.jpg" alt="Descrição">.

<div>

Usado para agrupar e estilizar conteúdo. Exemplo: <div class="container"></div>.

<p>

Define um parágrafo de texto. Exemplo: <p>Texto</p>.

<h1> a <h6>

Define cabeçalhos de diferentes níveis. Exemplo: <h1>Título Principal</h1>.

<span>

Usado para aplicar estilos a partes do texto. Exemplo: <span style="color:red">Texto</span>.

<ul>, <ol>, <li>

Cria listas não ordenadas e ordenadas. Exemplo: <ul><li>Item</li></ul>.

Comandos CSS

color

Define a cor do texto. Exemplo: color: red;.

background-color

Define a cor de fundo de um elemento. Exemplo: background-color: blue;.

font-size

Altera o tamanho da fonte. Exemplo: font-size: 20px;.

margin

Define a margem ao redor de um elemento. Exemplo: margin: 10px;.

padding

Define o espaço interno ao redor do conteúdo de um elemento. Exemplo: padding: 15px;.

border

Adiciona uma borda ao redor de um elemento. Exemplo: border: 1px solid black;.

display

Define o tipo de caixa de um elemento. Exemplo: display: block; ou display: inline;.

position

Controla o posicionamento de um elemento. Exemplo: position: absolute; ou position: relative;.

width e height

Define a largura e a altura de um elemento. Exemplo: width: 100px; e height: 50px;.

flexbox

Usado para criar layouts flexíveis. Exemplo: display: flex;.

grid

Usado para criar layouts baseados em grades. Exemplo: display: grid;.
